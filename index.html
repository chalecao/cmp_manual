Easily Learn & Use

学习成本很低，使用起来很方便，功能强大。通过简单的设计就能导出和视觉稿一模一样的组件，应用在工程项目中。支持功能定制与扩展，目前还在试用阶段！

Model Design & Reuse

代码编写的过程更像是设计的过程，设计中最本质的是设计数据模型，所有展示的元素都抽象成数据保存在内容中，你可以轻易的导出和导入，方便修改和重用！

Export Anything

最强大的功能是导出功能，同样的数据模型可以导出成不同语言所需要的组件，目前支持导出FTL组件，Regular组件和纯静态的页面。其他语言也容易定制扩展！

Think more, Do Best!

想得多，不如做的多。CMP几乎可以设计任何网页，它的整体设计思想是化繁为简，大事化小，小事化了。

  • Container && Function

    CMP引入了自治的思想，所以用Container控件画圈圈；引入了Function编程思想，语言是可以扩展的，范式是不变的。

  • User Centric Design

    前端编程更像是做设计，快速上手、敏捷实践。不仅面向开发人员，交互、视觉人员也可以快速掌握，视觉可以采用该软件制作视觉稿，开发只需要填充业务逻辑即可。

  • Model && ReUse

    CMP中的数据模型都是经过抽象独立的，与页面展示效果和行为相互独立。可以单独保存组件作为共享单元，高可重用性。

Enjoy Code with CMP!

CMP是一个工具，主要用来帮助您生成、管理各种组件/UI，通过简单的操作可以处理几乎所有和视觉稿相关的部分，并能实现75%以上的业务逻辑，通过数据抽象达到更深层次的UI复用。

CMP是从复杂的业务劳动中抽象、总结、开发出来的工具，主要是为了减少重复性的简单劳动，让开发人员更专注于处理业务逻辑，而不是浪费时间在编写CSS、HTML上，如果视觉人员是基于CMP制作视觉稿，开发人员能保证100%的视觉还原效果。
